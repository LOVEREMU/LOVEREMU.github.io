# EVA disscuss




  • “我的梦在哪里？”
  • “那是现实的延续”
  • “那，我的现实在哪里？”
  • “那是梦的终结”

YOU ARE (NOT) ALONE
YOU CAN (NOT) ADVANCE
YOU CAN (NOT) REDO
YOU MUST (NOT) RUN AWAY

人类的敌人，最终还是人类。
这一次，我一定要让你幸福。
这里，就是人类最后的阵地。
逃离了痛苦之后还会觉得痛苦。
对于我来说，生与死都是等价的。
也许我就是为了与你相遇才出生的。
梦是现实的延续，现实是梦的终结。
所谓的奇迹，要真的发生才有价值。
自己的死，那是唯一且绝对的自由。
讨厌自己的人不能够喜欢和信任别人。
我们只有以为自己幸福才能够活下去吗？
所谓长大成人，就是不断聚了散，散了聚。
人类因为畏惧黑暗，所以用火消除黑暗以求生存。
对不起，在这种时候，我不知道该用什么表情来面对你。
人有时就像刺猪一样，越是靠近彼此，越会伤害到对方。
人是无法完全理解别人的，连是否理解自己都值得怀疑。
不能逃避。不能逃避。不能逃避。不能逃避。不能逃避。
生与死的价值是同等，只有自己选择的死亡才是真正的自由。
人与人之间是无法完全相互了解的。人类、就是这么悲哀的生物。
时钟的针不会转回到原来的地方，不过事情是可以靠自己的力量去进行的。
只要有活下去的意念的话，任何地方都会是天国。因为还活着，幸福的机会就会有很多的。
人，是要靠忘掉回忆才能活下去的。但有些事，是绝对不能忘记的。只要活着，哪里都是天堂。
所谓长大成人，就是不断的聚了散，散了又聚，为了让彼此不会受伤害而保持一个适当的距离！
你只是渴望得到自己希望得到的东西而已，想得到幸福，不能只是等待，这样只能得到虚假的幸福。
无法传递给你的声音，再也触不到的思念，就让我一个人带走吧，希望下一个我，可以只爱上你一个人。
能够从这么多人之中成为 eva 的操纵者，然后与使徒战斗将它打败，而借此得到大家认同的话，这就是我最幸福的事。
生是死的开始，死是现实的延续，而再生是梦的终结。喜欢我吗？微笑是虚伪的，真实是痛苦的。溶合的心，将我毁灭。
每个人所持的世界观是很狭隘的。每个人都只会用自己的 ` 那把小尺子来量度事物，不客观的去看，只会按照被给予的他人的真实来看待事物。
人类常常会感到心痛，就是因为心痛，所以会觉得活着是一件很痛苦的事，尤其是你的心纤细、脆弱的有如玻璃一般，令人很有好感，所谓的好感就是喜欢。
不过，反正它... 没人照顾，早晚也是死。没有父母亲，也没有吃的，这种地方除了你跟我以外，应该也不会有人来，它还不是一样要活活饿死。所以到不如现在就杀了它嘛！
虽然我们现在一无所有，只要活着的话，总有一天，一定会觉得，能够活着是一件挺不错的事；虽然距离这一天或许还很遥远，不过，在那天来临之前，我们还是活下去吧。
下次分离的时候不要再说‘再见’这种悲伤的话了，虽然我们现在除了驾驶 Eva 之外一无所有，不过，只要活着的话，总有一天，一定会觉得，能够活着是一件不错的事。虽然距离这一天或许还很遥远，不过在那一天来临之前，我们还是活下去吧。
