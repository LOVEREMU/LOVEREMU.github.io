# 快速排序

时间复杂度：(nlogn)

# 思路

  1. 确定分界点
  2. 调整区间 (重点)；
    1. 设置左右指针
    2. 指针均向中间走，i 遇到比目标 x 小的就继续向下直到遇到比 x 大的就停下，j 反向走过来，同样遇到比 x 小的就停下，然后交换 i，j 所指向的数
    3. 继续向下，知道两指针相遇
  3. 递归处理左右两段

# 思路
