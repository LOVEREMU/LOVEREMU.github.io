# 面向对象编程

# 类、对象、实例

  • 类是 Java 中的一个重要概念，它是一种模板或蓝图，描述了对象应有的属性和方法
  • 对象是类的一个具体实例，可以通过类实例化出来。
  • 实例是对象的一个更具体的概念，表示某一具体的对象。

# 对象

对象是类的一个实例（对象不是找个女朋友），有状态和行为。例如，一条狗是一个对象，它的状态有：颜色、名字、品种；行为有：摇尾巴、叫、吃等。

# 类

类是一个模板，它描述一类对象的行为和状态。

# 继承

继承是 Java 中的一个重要概念，它允许一个类来继承另一个类的属性和方法。子类可以继承父类的公共属性和方法，这可以避免我们重复编写相同的代码，提高代码的复用性。

# 抽象

抽象是一种面向对象编程的思想，它允许我们声明一些没有具体实现的方法或类。在 Java 中，可以通过 abstract 来声明一个抽象方法或类。抽象方法只有方法的定义，没有具体的实现，而抽象类是指不能实例化的类，只能被用作其他类的父类。抽象类通常是某些类的共同基类，抽象方法则是该类子类必须实现的方法。

# 封装

封装是面向对象编程中的一个概念，它是指将属性和方法进行隐藏并限制访问的过程。在 Java 中，封装可以通过访问修饰符来实现。public 表示公共的，任何类都可以访问。private 表示私有的，只能在本类中访问。protected 表示受保护的，只能在本类和其子类中访问。封装可以有效地保护数据的安全性，并确保数据不会被意外修改。

# 多态

多态是面向对象编程的一个重要概念，它可以使得不同类型的对象都可以被当做同一类型来处理，从而使得代码更加灵活、可扩展。具体来说，多态包含两个方面的概念：重载和重写。

# 重载（Overloading）

是指在同一个类中定义多个方法，但这些方法拥有相同的方法名，但参数列表不同，在调用它们时，JVM 会根据参数列表自动判断调用哪个方法。

# 重写 （Overriding）

是指当一个子类继承自父类时，可以重写父类中的某个方法，以实现特殊的行为或改变现有行为。

# 方法、重载

方法是 Java 中的一个概念，用于描述一个类或对象可以执行的操作。方法可以被重载，即在同一个类中定义多个同名方法，但是它们的参数类型或个数不同，以适应不同的需求。重载可以通过方法名和参数类型来区分不同的方法，增加了代码的灵活性和可读性。
